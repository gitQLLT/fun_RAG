一个可以连接大模型和读取本地md文档内容，进行页面预览输入和输出，主要用于入门最快查看效果，用于装杯演示。

# 运行
安装依赖，需要 Python 3.10 以上版本。

```bash
pip install -r requirements.txt
gradio app.py
```

如果运行gradio报错：
```bash
python app.py
```
